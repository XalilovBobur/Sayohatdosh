# Sayohatdosh

**Sayohatdosh** — Toshkent shahri va Toshkent viloyatidagi qiziqarli, kam tanilgan va kashf etishga arziydigan joylarga sayohat qilish uchun hamroh topish platformasi. Loyiha foydalanuvchining qiziqishi, sayohat sanasi, budjeti, transport turi va guruh afzalligiga qarab mos yo‘nalishlar hamda sayohatdoshlarni topishga yordam beradi.

> **Slogan:** Hamroh top, yo‘lga chiq.

---

## Loyiha haqida

Sayohatdosh oddiy turistik katalog emas. Platformaning asosiy maqsadi — foydalanuvchiga Toshkent atrofidagi noodatiy joylarni topish, ularga borish uchun mos hamroh izlash va sayohatni xavfsizroq, qulayroq hamda qiziqarliroq qilish.

Loyiha avval **Sirli Sayohat**, **Ssayohat.com**, **Hamrohdosh** kabi nomlar bilan ishlab chiqilgan g‘oyalarga asoslanadi. Yakuniy brend yo‘nalishi sifatida **Sayohatdosh** tanlangan, chunki u o‘zbekcha, esda qoladigan va loyihaning asosiy g‘oyasini — sayohat + hamroh/topish — aniq ifodalaydi.

---

## Asosiy maqsad

- Toshkent va Toshkent viloyatidagi qiziqarli joylarni foydalanuvchiga ko‘rsatish.
- Yolg‘iz sayohat qilmoqchi bo‘lmagan foydalanuvchiga mos hamroh topish.
- Qiziqish, budjet, sana, transport va guruh turiga qarab sayohatlarni filtrlash.
- Foydalanuvchi uchun minimal, premium va mobilga mos interfeys yaratish.
- Admin panel orqali joylar, kategoriyalar va sayohat postlarini qulay boshqarish.

## Asosiy modellar konsepti

### Category

```python
class Category(models.Model):
    name = models.CharField(max_length=120)
    slug = models.SlugField(unique=True)
    icon = models.CharField(max_length=80, blank=True)
    description = models.TextField(blank=True)
    image = models.ImageField(upload_to="categories/", blank=True, null=True)
    is_active = models.BooleanField(default=True)

    def __str__(self):
        return self.name
```

### Location

```python
class Location(models.Model):
    category = models.ForeignKey(Category, on_delete=models.SET_NULL, null=True, related_name="locations")
    title = models.CharField(max_length=180)
    slug = models.SlugField(unique=True)
    description = models.TextField()
    region = models.CharField(max_length=120)
    address = models.CharField(max_length=255, blank=True)
    image = models.ImageField(upload_to="locations/", blank=True, null=True)
    price_from = models.PositiveIntegerField(default=0)
    rating = models.DecimalField(max_digits=3, decimal_places=2, default=0)
    latitude = models.DecimalField(max_digits=9, decimal_places=6, blank=True, null=True)
    longitude = models.DecimalField(max_digits=9, decimal_places=6, blank=True, null=True)
    is_popular = models.BooleanField(default=False)
    is_active = models.BooleanField(default=True)
    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    def __str__(self):
        return self.title
```

### TravelBuddyPost

```python
class TravelBuddyPost(models.Model):
    GROUP_CHOICES = [
        ("solo", "Yakka"),
        ("couple", "1–2 ta sherik"),
        ("family", "Oilaviy"),
        ("group", "Guruh"),
    ]

    TRANSPORT_CHOICES = [
        ("any", "Farqi yo‘q"),
        ("car", "Avtomobil"),
        ("taxi", "Taxi"),
        ("bus", "Avtobus"),
        ("walk", "Piyoda"),
    ]

    user = models.ForeignKey(User, on_delete=models.CASCADE, related_name="travel_posts")
    location = models.ForeignKey(Location, on_delete=models.CASCADE, related_name="travel_posts")
    travel_date = models.DateField()
    preferred_group = models.CharField(max_length=30, choices=GROUP_CHOICES, default="solo")
    transport_type = models.CharField(max_length=30, choices=TRANSPORT_CHOICES, default="any")
    budget = models.PositiveIntegerField(default=0)
    description = models.TextField(blank=True)
    is_active = models.BooleanField(default=True)
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.user} - {self.location}"
```

## O‘rnatish

### 1. Loyihani yuklab olish

```bash
git clone https://github.com/XalilovBobur/Sayohatdosh.git
cd Sayohatdosh
```

### 2. Virtual environment yaratish

Windows uchun:

```bash
py -m venv venv
venv\Scripts\activate
```

Agar PowerShell aktivatsiyada xatolik bersa:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
venv\Scripts\activate
```

### 3. Kutubxonalarni o‘rnatish

```bash
pip install -r requirements.txt
```

### 4. `.env` fayl yaratish

`.env.example` asosida `.env` fayl yarating:

```env
SECRET_KEY=your-secret-key
DEBUG=True
ALLOWED_HOSTS=127.0.0.1,localhost
DATABASE_URL=sqlite:///db.sqlite3
```

### 5. Migratsiyalarni bajarish

```bash
python manage.py makemigrations
python manage.py migrate
```

### 6. Superuser yaratish

```bash
python manage.py createsuperuser
```

### 7. Serverni ishga tushirish

```bash
python manage.py runserver
```

Brauzerda oching:

```text
http://127.0.0.1:8000/
```

## Muhim template namespace misollari

Template ichida URL chaqirishda namespace xatolarini oldini olish uchun quyidagi uslubdan foydalaniladi:

```django
<a href="{% url 'core:home' %}">Bosh sahifa</a>
<a href="{% url 'locations:list' %}">Joylar</a>
<a href="{% url 'travel_buddies:list' %}">Hamroh topish</a>
<a href="{% url 'users:login' %}">Kirish</a>
```

Agar app nomi boshqacha bo‘lsa, template ichidagi namespace ham aynan `urls.py` dagi `app_name` bilan bir xil bo‘lishi kerak.

## Muammo va yechimlar

### Rasmlar chiqmayapti

Tekshiring:

- `MEDIA_URL` sozlanganmi?
- `MEDIA_ROOT` sozlanganmi?
- `urls.py` ichida `static(settings.MEDIA_URL, ...)` qo‘shilganmi?

### URL namespace xatoliklari

Tekshiring:

- har bir app `urls.py` ichida `app_name` borligini;
- template ichida `app_name:url_name` to‘g‘ri yozilganini;
- project `urls.py` ichida app `include()` qilinganini.

Tekshiring:

- `{% load static %}` yozilganmi?
